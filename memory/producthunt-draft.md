# coderev — Product Hunt 发布准备

## 基本信息

- **产品名称**: coderev — Multi-Agent AI Code Review CLI
- **标语 (Tagline)**: Three AI agents review your PRs in parallel — Security, Bug, and Quality — with confidence scoring
- **一句话描述**: Open-source CLI tool that runs 3 parallel AI agents on every code review, scoring issues by confidence
- **类别**: Developer Tools / Open Source

## 详细描述 (Description)

> coderev is a CLI tool that transforms code review with multi-agent AI. When you run `coderev review`, it launches three specialized AI agents in parallel:
>
> - 🔒 **Security Agent** — OWASP-level vulnerability scanning (SQL injection, XSS, SSRF, hardcoded secrets)
> - 🐛 **Bug Agent** — Null pointers, race conditions, async issues, logic errors
> - 📐 **Quality Agent** — Code complexity, DRY violations, naming, error handling
>
> Each issue gets a **confidence score 0-100**. Low-confidence false positives are automatically filtered. You get a clear, actionable report — not noise.
>
> ### Key Features
>
> - 🚀 **Parallel multi-agent review** — 3x faster than sequential analysis
> - 📊 **Confidence scoring** — Focus on what matters, ignore false positives
> - 🛠 **Interactive fixing** — Apply AI-suggested fixes with one click
> - 📋 **HTML reports** — Beautiful visual reports with dark/light themes
> - 🔄 **CI/CD ready** — Works with GitHub Actions, GitLab CI, and any CI pipeline
> - 🌐 **Multi-platform** — GitHub, GitLab, Gitee, Bitbucket, GitCode
> - 💡 **Git blame context** — Know which issues are new vs pre-existing
> - ⚙️ **Config inheritance** — Team config at root, project overrides in subdirs
>
> ### Who is it for?
>
> - **Solo developers** — Get a second set of AI eyes before every commit
> - **Open source maintainers** — AI pre-screens PRs so you focus on high-impact reviews
> - **Teams** — Consistent code quality standards with shared configuration
> - **CI/CD pipelines** — Block risky code before it ships
